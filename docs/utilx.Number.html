<static> mod(a, b) → {number}

The mod/remainder operator returns the first operand modulo of the second operand, that is, number1 modulo number2, in the preceding statement, where number1 and number2 are numbers. The modulo function is the integer remainder of dividing number1 by number2. For example, 12 % 5 returns 2. The result will have the same sign as number1; that is, -1 % 2 returns -1.
